## 🔧 Common Issues and Fixes

### ❌ "Failed to Attach" on Injection
**Cause:** Roblox updated after your current delta build, or antivirus is blocking the loader.
**Fix:** Restart delta — the auto-updater fetches the latest patch on launch. If still failing, add the delta folder to your antivirus exclusion list and run as Administrator.

---

### ❌ Script Executes but Has No In-Game Effect
**Cause:** The script targets a game feature that has since been patched, or it was written for an older Roblox API.
**Fix:** Test a simple Script Hub entry (speed script, jump script) to confirm the executor is working. If Hub scripts execute correctly, the problem is with the specific script — try an updated version.

---

### ❌ Delta Crashes at Launch (Windows)
**Cause:** Missing .NET 6.0 runtime or outdated Visual C++ Redistributable.
**Fix:** Install the latest [Microsoft .NET Runtime](https://dotnet.microsoft.com/download) and the [Visual C++ Redistributable 2022 (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe), then relaunch as Administrator.

---

### ❌ Android APK Won't Install
**Cause:** "Install Unknown Apps" not enabled, or not enough free storage.
**Fix:** Settings → Security → enable Install Unknown Apps for your file manager. Free at least 250 MB of internal storage before installing the **delta executor download apk**.

---

### ❌ iOS Build Shows "Untrusted Developer"
**Cause:** Developer certificate not trusted, or the IPA has expired after 7 days (free Apple Developer limit).
**Fix:** Settings → General → VPN & Device Management → delta certificate → **Trust**. If expired, repeat the full sideload process with the current **delta executor ipa** from the Releases section.

---

### ❌ macOS Build Blocked by Gatekeeper
**Cause:** macOS quarantine flag applied to unsigned `.app` bundle.
**Fix:** Open Terminal and run: `xattr -cr /Applications/delta.app` — this clears the quarantine attribute without disabling Gatekeeper system-wide.

---

### ❌ Script Hub Loads Blank
**Cause:** Internet connection issue or Script Hub servers undergoing maintenance update.
**Fix:** Check your network connection. If online, wait 5–10 minutes and reopen the Hub tab. Maintenance windows are announced in the **delta executor discord server** `#status` channel.
